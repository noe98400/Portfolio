Table mit Attributes

Name Beschreibung
rel Definiert die Beziehung zwischen der verlinkten Source und dem aktuellen Dokument.
alt Gehört zum Tag img. Dort wird ein Text definiert, der angezeigt wird, falls das Bild nicht geladen werden kann
controls Gehört zum Tag video. Macht, dass es ein Play und Pause Knopf hat.
target Gehört zum Tag Link: definiert, wie die angeklickte Website geöffnet werden soll. Z.B _blank: die Seite wird in einem neuen Tab aufgemacht.
